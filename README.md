# blog

## このリポジトリについて

フロントエンドのリハビリのため、Vue.jsとFirebaseを使ってはてなダイアリーの更新ができなくなった事で行き先を失ったブログ記事置き場を作ろうという試み。

これも別にソースコードのライセンスは不要なのでMITにしたい。
